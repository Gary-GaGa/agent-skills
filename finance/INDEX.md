# Finance

Personal investing, market analysis, and trading discipline.

## Skills (9)

| Skill | Description |
|-------|-------------|
| [`portfolio-construction`](./portfolio-construction/SKILL.md) | 投資組合的報酬有 90% 由資產配置決定，不是由選股決定。先決定「放多少在股票、多少在現金」，再決定「買哪些股票」。 |
| [`tw-etf-investing`](./tw-etf-investing/SKILL.md) | ETF 的核心價值是分散風險和低成本。不用選股、不用盯盤，長期跑贏多數主動投資者。 |
| [`tw-stock-chip`](./tw-stock-chip/SKILL.md) | 籌碼面回答的是：「這檔股票被誰拿在手上？這些人想買還是想賣？」基本面判斷「值不值得買」，籌碼面判斷「現在能不能買」。 |
| [`tw-stock-data`](./tw-stock-data/SKILL.md) | 垃圾進，垃圾出。量化投資的品質上限 = 資料品質的上限。花 80% 的時間處理數據是正常的。 |
| [`tw-stock-fundamental`](./tw-stock-fundamental/SKILL.md) | 基本面回答的是：「這家公司值不值得擁有？」籌碼面回答的是：「現在是不是進場的好時機？」兩者互補，不互斥。 |
| [`tw-stock-options`](./tw-stock-options/SKILL.md) | 選擇權是權利，不是義務。買方付權利金買一個「選擇的權利」；賣方收權利金承擔「被執行的義務」。 |
| [`tw-stock-quant`](./tw-stock-quant/SKILL.md) | 量化的核心不是寫程式，是把「我的投資邏輯」轉換成可驗證、可重複、可改進的規則。程式只是執行工具。 |
| [`tw-stock-tax`](./tw-stock-tax/SKILL.md) | 稅是確定的成本。不了解稅，你的實際報酬率可能比想像中低很多。 |
| [`tw-stock-technical`](./tw-stock-technical/SKILL.md) | 技術分析回答的是：「現在的價格走勢暗示什麼？」它不告訴你公司好不好（基本面），也不告訴你誰在買（籌碼面），它告訴你「市場共識正往哪個方向移動」。 |
