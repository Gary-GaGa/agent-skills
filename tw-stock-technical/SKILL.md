# 台股技術分析

> 技術分析回答的是：「現在的價格走勢暗示什麼？」它不告訴你公司好不好（基本面），也不告訴你誰在買（籌碼面），它告訴你「市場共識正往哪個方向移動」。

## 適用情境

- 已有基本面 + 籌碼面的候選名單，需要判斷具體進出場點位
- 觀察盤面走勢，判斷短線（1-2 週）趨勢方向
- 設定停損停利的技術位置
- 判讀大盤加權指數的多空格局
- 避免在技術面弱勢時逆勢進場

## 一、K 線基礎

### 單根 K 線

| 型態 | 形狀特徵 | 意義 |
|------|----------|------|
| **大紅 K（長陽線）** | 實體長、上下影線短 | 多方強勢掌控 |
| **大黑 K（長陰線）** | 實體長、上下影線短 | 空方強勢掌控 |
| **十字線** | 實體極短、上下影線等長 | 多空拉鋸，轉折訊號 |
| **上影線長** | 上影線 > 實體 2 倍以上 | 上方賣壓重（射擊之星） |
| **下影線長** | 下影線 > 實體 2 倍以上 | 下方買盤強（鎚子線） |

**台股注意：** 台股 K 線紅漲黑跌（與美股相反），看技術分析書要注意紅黑定義。

### 常見 K 線組合

| 型態 | 出現位置 | 意義 |
|------|----------|------|
| **晨星（Morning Star）** | 底部 | 下跌 → 十字 → 上漲，底部反轉訊號 |
| **夜星（Evening Star）** | 頂部 | 上漲 → 十字 → 下跌，頭部反轉訊號 |
| **吞噬（Engulfing）** | 頂/底 | 第二根完全包覆第一根，強力反轉 |
| **紅三兵** | 底部 | 連三紅 K 逐步墊高，多方蓄勢 |
| **黑三鴉** | 頂部 | 連三黑 K 逐步走低，空方發動 |
| **島狀反轉** | 頂/底 | 跳空上去又跳空下來（或反之），強力反轉 |

### K 線使用原則

1. **不要單看一根 K 線決策。** K 線是「線索」不是「結論」。
2. **搭配位置（支撐/壓力）和量能判讀才有意義。** 在半山腰出現十字線 ≠ 在底部出現十字線。
3. **日 K 看短線、週 K 看中線、月 K 看長線。** 不同時間框架的 K 線訊號可能矛盾——以大框架為主。

## 二、均線系統

### 常用均線

| 均線 | 代碼 | 性質 | 代表 |
|------|------|------|------|
| 5 日 | MA5 | 短線 | 一週均價（極短線操盤線）|
| 10 日 | MA10 | 短線 | 兩週均價 |
| 20 日 | MA20 | 短中線 | 月線（散戶最常看）|
| 60 日 | MA60 | 中線 | 季線（法人成本線）|
| 120 日 | MA120 | 中長線 | 半年線 |
| 240 日 | MA240 | 長線 | 年線（長線多空分界）|

### 均線排列

| 排列 | 形態 | 意義 |
|------|------|------|
| **多頭排列** | MA5 > MA20 > MA60 > MA120 > MA240 | 強勢上漲趨勢 |
| **空頭排列** | MA5 < MA20 < MA60 < MA120 < MA240 | 強勢下跌趨勢 |
| **糾結** | 各均線纏繞在一起 | 即將變盤，等方向確認 |

### 均線訊號

| 訊號 | 意義 | 補充 |
|------|------|------|
| **黃金交叉** | 短均線從下穿越長均線 | 多頭啟動，但要確認量能配合 |
| **死亡交叉** | 短均線從上穿越長均線 | 空頭啟動 |
| **股價站上月線** | 收盤價 > MA20 | 短線轉強的第一步 |
| **股價跌破季線** | 收盤價 < MA60 | 中線轉弱，法人成本線失守 |
| **股價跌破年線** | 收盤價 < MA240 | 長線多空反轉警訊 |

### 均線使用原則

4. **均線是落後指標。** 它確認趨勢，不預測轉折。
5. **突破或跌破後，觀察「站穩」還是「假突破」。** 連續 3 日站上/跌破比較可靠。
6. **在多頭排列中找回檔買點（回測均線不破就是支撐），在空頭排列中找反彈賣點。**

## 三、技術指標

### MACD（指數平滑異同移動平均線）

- **組成：** DIF 線、MACD 線（Signal）、柱狀體（Histogram）
- **預設參數：** 12, 26, 9

| 訊號 | 意義 |
|------|------|
| DIF 上穿 MACD（金叉） | 多方動能轉強 |
| DIF 下穿 MACD（死叉） | 空方動能轉強 |
| 柱狀體由負轉正 | 跌勢趨緩，可能反轉 |
| 柱狀體連續縮短 | 現有趨勢力道減弱 |
| **背離**（股價新高 but DIF 沒新高）| 頂背離 → 即將回檔 |
| **背離**（股價新低 but DIF 沒新低）| 底背離 → 即將反彈 |

**重點：** MACD 的背離是所有技術指標中最強的反轉訊號之一。

### RSI（相對強弱指標）

- **範圍：** 0～100
- **預設：** RSI(14)

| 區間 | 意義 |
|------|------|
| RSI > 80 | 超買，短線回檔風險高 |
| RSI < 20 | 超賣，短線反彈機會大 |
| RSI 50 上下 | 多空分界，看趨勢方向 |
| RSI 背離 | 與 MACD 背離相同邏輯 |

**注意：** 強勢股可以在 RSI 80 以上維持數週。超買 ≠ 一定會跌，只代表「漲太快」。

### KD（隨機指標）

- **組成：** K 線、D 線（K 的平滑）
- **範圍：** 0～100
- **預設：** 9, 3, 3

| 訊號 | 意義 |
|------|------|
| K 上穿 D（金叉） | 短線買進訊號 |
| K 下穿 D（死叉） | 短線賣出訊號 |
| KD > 80 | 超買區 |
| KD < 20 | 超賣區 |
| 低檔鈍化（KD < 20 持續） | 超賣但賣壓持續，不宜搶反彈 |
| 高檔鈍化（KD > 80 持續） | 超買但買氣持續，不宜摸頭放空 |

**KD vs RSI：** KD 更靈敏（適合短線），RSI 更穩定（適合中線）。

### 布林通道（Bollinger Bands）

- **組成：** 中線（MA20）、上軌（+2 標準差）、下軌（-2 標準差）
- 統計上約 95% 的價格會在通道內

| 型態 | 意義 |
|------|------|
| **股價觸碰上軌** | 短線過熱，但強勢股會沿上軌走（不代表賣） |
| **股價觸碰下軌** | 短線超跌，但弱勢股會沿下軌走（不代表買） |
| **通道收窄（縮口）** | 波動率降低，即將變盤。方向未定。 |
| **通道突然擴張** | 趨勢發動，跟上突破方向 |
| **股價跌出下軌又收回** | 假跌破，短線反彈訊號 |

## 四、量價關係

### 核心原則

**量是價的先行指標。** 沒有成交量配合的價格變動不可靠。

| 量價組合 | 意義 |
|----------|------|
| **價漲量增** | 健康上漲，多方有信心 |
| **價漲量縮** | 追價動能不足，漲勢可能到頂 |
| **價跌量增** | 恐慌性賣壓，但也可能是洗盤（看位置） |
| **價跌量縮** | 賣壓減緩，可能接近底部 |
| **爆量長紅** | 突破訊號（在盤整底部出現最有意義） |
| **爆量長黑** | 多殺多出貨（在高檔出現最危險） |
| **量平價漲** | 無量上漲，持續性存疑 |

### 量的基準

7. **「爆量」= 成交量 > 20 日均量的 2 倍以上。**
8. **「量縮」= 成交量 < 20 日均量的 50% 以下。**
9. **觀察相對量，不要看絕對張數。** 台積電日均量 3 萬張和小型股日均量 300 張沒有可比性。

## 五、支撐與壓力

### 支撐（買盤容易出現的價位）

| 來源 | 說明 |
|------|------|
| **前波低點** | 跌到前次低點附近會有人抄底 |
| **均線** | MA20、MA60、MA240 都是動態支撐 |
| **跳空缺口上緣** | 缺口不補 = 強支撐 |
| **整數關卡** | 心理關卡（100 元、500 元） |
| **量密集區下緣** | 大量成交的區間底部 |

### 壓力（賣盤容易出現的價位）

| 來源 | 說明 |
|------|------|
| **前波高點** | 漲到前次高點附近會有人停利 |
| **均線** | 空頭排列時均線是壓力 |
| **跳空缺口下緣** | 向下跳空的缺口是壓力 |
| **套牢區** | 前波大量成交的橫盤區域（一堆人被套，解套就跑） |

### 支撐/壓力互換原則

10. **支撐被跌破後變壓力；壓力被突破後變支撐。** 這是技術分析最核心的規則之一。

## 六、型態學（Price Patterns）

### 反轉型態

| 型態 | 出現位置 | 意義 |
|------|----------|------|
| **頭肩頂** | 頂部 | 左肩-頭-右肩，跌破頸線確認下跌 |
| **頭肩底** | 底部 | 反向，突破頸線確認上漲 |
| **雙重頂（M 頭）** | 頂部 | 兩次碰到壓力都上不去 |
| **雙重底（W 底）** | 底部 | 兩次碰到支撐都跌不破 |
| **圓弧頂/底** | 頂/底 | 緩慢反轉，能量漸變 |

### 持續型態

| 型態 | 意義 |
|------|------|
| **旗形** | 趨勢中的小幅反向整理，完成後繼續原方向 |
| **三角收斂** | 高低點漸收窄，突破方向即趨勢方向 |
| **箱型整理** | 在一個區間內橫盤，上破做多、下破做空 |

### 型態使用原則

11. **所有型態都要「確認」才成立。** 頭肩頂要跌破頸線、三角要突破邊界。未確認前只是「疑似」。
12. **量能配合。** 反轉型態完成時，突破方向應伴隨放量。
13. **時間框架。** 週線的型態 > 日線的型態（週線雙重底比日線雙重底可靠得多）。

## 七、多指標共振框架

**單一指標容易騙線。多指標同時指向同一方向才有意義。**

### 進場訊號（至少 3/5 成立）

- [ ] 趨勢：股價在月線之上，均線呈多頭排列（或至少不是空頭排列）
- [ ] 量能：近期出現過爆量長紅突破，或量價配合健康
- [ ] 指標：MACD 金叉或柱狀體翻正；RSI > 50
- [ ] 型態：在支撐位附近出現底部反轉 K 線組合
- [ ] 背離：沒有頂背離（MACD 或 RSI）

### 出場 / 警戒訊號（出現 2/4 即警戒）

- [ ] 股價跌破月線且 3 日未站回
- [ ] MACD 死叉或出現頂背離
- [ ] 爆量長黑出現
- [ ] 跌破關鍵支撐（前低、缺口、頸線）

### 不同時間框架整合

| 大框架（週線） | 小框架（日線） | 策略 |
|----------------|----------------|------|
| 多頭 | 多頭 | **積極做多** |
| 多頭 | 回檔 | **等回檔結束再進場**（最好的買點） |
| 空頭 | 反彈 | **觀望或短線搶反彈**（風險高） |
| 空頭 | 空頭 | **空手等待** |

14. **大框架決定方向，小框架決定時機。** 不要在週線空頭中因為日線金叉就重倉。

## 八、常見陷阱

| 陷阱 | 說明 | 對策 |
|------|------|------|
| **假突破** | 股價衝過壓力又收回 | 等收盤確認 + 隔日站穩 |
| **指標鈍化** | KD/RSI 在超買/超賣區長時間不回 | 強勢/弱勢股鈍化是正常的，不要逆勢 |
| **只看一個指標** | RSI 超賣就買、MACD 金叉就買 | 多指標共振才下手 |
| **忽略大盤** | 個股技術面再好，大盤崩也逃不了 | 先看大盤趨勢再看個股 |
| **過度看盤** | 每分鐘盯 K 線，情緒決策 | 設好進出場條件就離開 |
| **後視鏡偏差** | 回頭看「這裡這麼明顯」 | 只根據當下可見的資訊決策 |
| **預測而非跟隨** | 「我覺得要反轉了」 | 等訊號出現再行動 |

## 九、技術分析檢查清單

每次看盤做判斷前：

- [ ] 大盤加權指數的趨勢方向（月線之上或之下）
- [ ] 個股所在時間框架：上漲/下跌/盤整？
- [ ] 均線排列狀態（多頭/空頭/糾結）
- [ ] 最近的支撐位和壓力位在哪裡
- [ ] 量能配合度（有量的漲 vs 無量的漲）
- [ ] MACD 方向（金叉/死叉）與是否有背離
- [ ] RSI/KD 所在區間（正常/超買/超賣）
- [ ] 布林通道寬窄（即將變盤或正在趨勢中）
- [ ] 有無完成的型態（頭肩頂、W 底等）
- [ ] 與基本面 / 籌碼面是否一致

## 十、技術分析與其他分析面的整合

| 基本面 | 籌碼面 | 技術面 | 判斷 |
|--------|--------|--------|------|
| 好 | 好 | 好 | **最佳進場點** |
| 好 | 好 | 差 | 等技術面轉強（回測支撐站穩）|
| 好 | 差 | 好 | 技術面可能只是短線反彈，謹慎 |
| 差 | — | 好 | 純技術面操作，短線思維、嚴守停損 |

**三面共振是最高勝率的進場訊號。** 任何一面不支持，就降低部位或等待。
